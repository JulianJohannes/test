Detailed Description

Calculates the TotalVaration of a field \psi and its absolute and relative error compared to the initial state. The calculates properties are written to a CSV file "TVerror.csv".

<equation>

Usage
Minimal example by using system/controlDict.functions: 
TVerrorCSV
{
    type            TVerrorCSV;
    libs            ( leiaFunctionObjects );
    writeControl    timeStep;
    field           psi;
}
