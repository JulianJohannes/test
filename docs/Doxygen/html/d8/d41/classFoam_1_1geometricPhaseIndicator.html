Detailed Description

Computes the phase indicator as a volume fraction from a field \psi_c.

In each cell \Omega_c, a piecewise-planer approximation \psi^l_c is computed using linear least squares. This implicit plane is then used to truncate the cell and compute the phase-specific volume \Omega_c^-, resulting in the volume fraction

\alpha_c = |\Omega_c^-| / |\Omega_c|
