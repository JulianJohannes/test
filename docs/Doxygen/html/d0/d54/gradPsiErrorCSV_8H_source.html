32 Description
33  Looks up a signedDistance-error field (from gradPsiError functionObject for example).
34  Calculates further properties and writes them to a CSV file "gradPsiError.csv"
35  The following properties a written in every row.
36  "TIME,"
37  "MAX_DELTA_X,"
38  "MEAN_DELTA_X,"
39  "E_MAX_GRAD_PSI,"
40  "E_MEAN_GRAD_PSI,"
41  "E_NARROW_MAX_GRAD_PSI,"
42  "E_NARROW_MEAN_GRAD_PSI\n"
43

46  \verbatim
47  gradPsiErrorCSV
48  {
