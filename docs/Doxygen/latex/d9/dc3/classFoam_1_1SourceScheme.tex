\doxysubsection{Detailed Description}
The base class \mbox{\hyperlink{classFoam_1_1SourceScheme}{Source\+Scheme}} forms the root for the discretization hierarchy of the SDPLS source term implemented in \mbox{\hyperlink{classFoam_1_1SDPLSSource}{SDPLSSource}}. 

\label{_Description}%


For the discretization, the non-\/linear SDPLS source term $S_{SDPLS}$ is linearized as \begin{equation} S_{SDPLS}(\psi^{n+1}) \approx S_c + S_{p} \psi^{n+1}_c, \end{equation} where $S_c$ is a constant and $S_{p}$ is the proportionality constant for the linear part. Initializing the discretization and returning the tmp$<$fv\+Scalar\+Matrix$>$ is done by the public method

discretize(non\+Linear\+Part, psi),

which takes the non\+Linear\+Part and the LS field psi from the source term implementing the class \mbox{\hyperlink{classFoam_1_1SDPLSSource}{SDPLSSource}}. Inheriting classes are responsible for implementing \mbox{\hyperlink{classFoam_1_1SourceScheme_a4d53deabbecc3fa7d747bc576d506e91}{update\+Sc()}} for $S_c$ and \mbox{\hyperlink{classFoam_1_1SourceScheme_afbedde408444a4d4d78913d3596885e8}{update\+Sp()}} for $S_p$, which are called by \mbox{\hyperlink{classFoam_1_1SourceScheme_afe5c06fbfaef382705462a2f32d8ffe0}{discretize()}}. This base class implements the case of no discretization with \begin{equation} S_{noSource}(\psi^{n+1})=0. \end{equation}
