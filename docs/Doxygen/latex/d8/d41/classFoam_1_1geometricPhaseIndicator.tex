\doxysubsection{Detailed Description}
Computes the phase indicator as a volume fraction from a field \textbackslash{}psi\+\_\+c. 

\label{_Description}%


In each cell \textbackslash{}\+Omega\+\_\+c, a piecewise-\/planer approximation \textbackslash{}psi$^\wedge$l\+\_\+c is computed using linear least squares. This implicit plane is then used to truncate the cell and compute the phase-\/specific volume \textbackslash{}\+Omega\+\_\+c$^\wedge$-\/, resulting in the volume fraction

\textbackslash{}alpha\+\_\+c = $\vert$\textbackslash{}\+Omega\+\_\+c$^\wedge$-\/$\vert$ / $\vert$\textbackslash{}\+Omega\+\_\+c$\vert$
