\doxysubsection{Detailed Description}
The class \mbox{\hyperlink{classFoam_1_1explicitScheme}{explicit\+Scheme}} implements the explicit discretization of the source term. 

\label{_Description}%


The linearization constants are calculated as \begin{equation} \begin{aligned} S_{explicit,\,c} &= f_{nl}(\psi^n) \psi^n_c\\ S_{explicit,\,p} &= 0 \end{aligned} \end{equation} where the constant $S_c$ explicitly evaluates the entire SDPLS source term. With this, the explicit source is written as \begin{equation} S_{explicit}(\psi^{n+1}) = f_{nl}(\psi^n) \psi^n \end{equation} Without any implicit part, this discretized source term only contributes to the r.\+h.\+s. of the resulting algebraic linear system.
