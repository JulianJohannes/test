27  Foam::explicitScheme
28 
29 Description
30  The class explicitScheme implements the explicit discretization of the source term.
31 
32  The linearization constants are calculated as
33  \f{equation}{
34  \begin{aligned}
35  S_{explicit,\,c} &= f_{nl}(\psi^n) \psi^n_c\\
36  S_{explicit,\,p} &= 0
37  \end{aligned}
38  \f}
39  where the constant \f$S_c\f$ explicitly evaluates the entire SDPLS source term. With this, the explicit source is written as
40  \f{equation}{
41  S_{explicit}(\psi^{n+1}) = f_{nl}(\psi^n) \psi^n
42  \f}
43  Without any implicit part, this discretized source term only contributes to the r.h.s. of the resulting algebraic linear system.
